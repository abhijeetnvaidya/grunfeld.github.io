C.A.S.E.O.

Cross All Squares Exactly Once

An open source clone of Alcazar

Objective

Enter the room from one of the doors on the border and exit from another.

Cross every square inside the room exactly once.

Game Mechanics

=    Use mouse as a brush to draw path between the squares.

   Re-brushing a drawn path/segment erases it.

   Use [Toggle-Lock] to save the progress.

   [Undo] erases all the paths drawn after the save point.
